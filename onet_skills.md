## How to Use This Data in Mentorable

**For roadmap generation:** Feed the Critical and High skill tiers directly into the Claude prompt as the `required_skills` field. Skills with importance ≥ 70 should appear as milestones. Skills between 55–69 should appear as supporting milestones.

**For skill gap analysis:** Compare student's self-reported skills against the Critical tier for their matched career. Any Critical skill they're missing becomes a priority milestone in Phase 1 or 2.

**For the profile radar chart:** The 5 skills with the highest importance scores for a given career define the radar chart axes on the student's profile reveal screen.

**For career matching:** A student who scores high on Programming + Complex Problem Solving + Active Learning maps strongly to Software Developer, Computer Programmer, or Web Developer. High Judgment + Science + Reading maps to Physician Assistant or Biomedical Engineer.

**Known gaps in this dataset:** Data Scientist (15-2051.00) and Financial Analyst (13-2051.00) returned no rows — these O*NET codes may not appear in the skill CSVs as exported. Supplement with manual O*NET lookup at onetonline.org for these two careers.
